# Contributors / 贡献账本

Jellyfish keeps a lightweight public contribution ledger so important work is visible and attributed.

Principle: **wide entrance, narrow trunk**. Many contributions can enter through issues, discussions, RFCs, and pull requests; only reviewed, redacted, reproducible, and safe work moves into the main trunk.

Do not record private raw data, secrets, or sensitive user details here. Link to public issues/PRs/RFCs when possible.

## How to add a contribution

Add accepted or important contributions under the most relevant type:

```md
- Contributor: @handle or name
  Contribution: short description
  Link: issue / PR / RFC / discussion
  Date: YYYY-MM-DD
```
