# Open Source Position / 开源立场

认知水母必须开源优先，因为它的目标不是单一公司平台，而是中立认知公地。

闭源能做产品，开源才能做文明接口。

## Open Protocols

Residual Cards, Skill Capsules, Proof of Compression, Human Transition Cards, Cognitive Fingerprint Rights, and Contribution Graphs should be auditable protocol surfaces.

## Private Data

Open source does not mean open memory. Personal memories, private caches, secrets, enterprise tickets, and sensitive source material must remain private by default.

## Open Core

The core protocols, mock gateways, safety gates, and governance docs should be open for review, critique, fork, and improvement.

## Commercial Services

Commercial services can provide hosting, deployment, plugins, model routing, team collaboration, safety audits, and UI experiences.

They must not privatize public protocols or the Public Skill Commons.

## Nonprofit Protocol Foundation

A future nonprofit protocol foundation can steward public protocols, safety standards, compatibility tests, and public commons governance.

## Community Governance

Community governance should be transparent, reversible where possible, and accountable to privacy, consent, human transition, and anti-abuse boundaries.
