# Cognitive Commons / 认知公地

The cognitive commons is not a public privacy pool.

It only accepts anonymized, authorized, verified, structured experience.

## Cache Layers

- Private Cache
- Team Cache
- Community Cache
- Public Skill Commons

## Core Principle

Raw data stays private.
Structured, anonymized, verified experience may become shared intelligence.

## What Can Enter the Commons

- Residual Cards with sensitive fields removed
- Skill Capsules with provenance and review state
- Proof of Compression
- Public-safe templates
- Public-safe world experience summaries
- Contribution records that do not expose private memory

## What Cannot Enter the Commons

- Raw private chats
- Student/customer identities
- Enterprise secrets
- API keys
- Unlicensed IP/persona data
- Minor data without strict privacy and consent controls
