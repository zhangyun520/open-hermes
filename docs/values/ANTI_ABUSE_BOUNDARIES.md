# Anti-Abuse Boundaries / 反滥用边界

Cognitive Jelly must not become an exploitation engine, gambling loop, privacy black box, cheating tool, or unauthorized persona/IP platform.

## Forbidden

- 共享账号
- 薅网页会员额度
- 交易 token
- 承诺收益
- 上传他人隐私
- 未授权 IP 复刻
- 真人冒用
- 外挂
- 代练
- 反作弊绕过
- 自动发送高风险外部消息
- 自动做保险/医疗/法律/金融承诺
- 把认知指纹当社会信用分
- 把贡献账本当功德审判

## Required Boundaries

- Private data remains private.
- Public cache requires anonymization, consent, review, and provenance.
- High-risk actions require human approval.
- Game/world connectors must not bypass anti-cheat.
- Persona and IP systems require authorization.
- Gameification must not become gambling.
- Contributions must not become moral rankings.
