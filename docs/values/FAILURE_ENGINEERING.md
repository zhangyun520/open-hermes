# Failure Engineering / 失败工程学

失败不是成功之母。
失败是未提纯的工程材料。

Failure Engineering studies how to recover, classify, anonymize, review, cache, and reuse failures until they become system capability.

## Flow

```text
Failure → Residual → Residual Card → Review → Skill Capsule → Cache Hit → Lower Future Cost
```

## What Counts as Failure Material

- Wrong answers
- Failed prompts
- Customer objections
- Enterprise tickets
- AI hallucinations
- Game experience breakdowns
- Narrative contradictions
- Human-transition harms

## Branches

- 错题工程学 / wrong-answer engineering
- 异议工程学 / objection engineering
- 工单工程学 / ticket engineering
- 幻觉工程学 / hallucination engineering
- 沉浸工程学 / immersion engineering
- 安置工程学 / transition engineering

## Principle

A failure should not be used to shame a person. It should be converted into a safer structure, a clearer residual, a better review path, or a reusable skill.
