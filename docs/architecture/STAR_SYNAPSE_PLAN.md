# Star-Synapse Plan / 星链突触计划

## 1. Mission

星链突触计划的目标不是发射物理卫星，而是建立一个分布式认知星座。

每一个用户设备，是一个微小感知点。
每一个团队缓存，是一个局部神经团。
每一个行业插件，是一个领域器官。
每一个 CPU 节点，是一个分解者。
每一个 GPU 节点，是一个突触铸造炉。
每一个专家节点，是一个价值感受器。
每一个公共技能池，是一个共享星云。
每一个开源协议，是这些星点之间的轨道规则。

## 2. Four Layers

第一层：边缘残差层

个人设备、本地水母、私有缓存、游戏体验、教育错题、工业工单、客户异议。

第二层：神经肥料层

脱敏、结构化、审核、验证后的 F2-F5 肥料。

第三层：突触铸造层

CPU/GPU/专家/公共节点生成小突触。

第四层：柔性模型层

模型积木、技能积木、路由积木、评测积木、安全积木组合成柔性智能。

## 3. Core Principles

- 原始数据默认私有。
- 不是所有残差都能训练。
- 先训练小突触，再考虑反哺大模型。
- 不要一个巨兽吞掉世界，而要让智能像神经系统一样柔性生长。
