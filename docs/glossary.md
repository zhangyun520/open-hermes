# Glossary / 术语表

## Jellyfish

An open co-creation system for human-AI symbiosis, also called Cognitive Jelly / 认知水母 in this prototype.

## Wide entrance, narrow trunk

A governance pattern where many contribution types can enter, but only reviewed, redacted, reproducible, and safe work becomes canonical.

## Residual / D

The unresolved gap between current closure and needed closure. In the project formula, `C + D = 1`.

## Closure / C

The degree to which a need, task, failure, or workflow has been understood, verified, and made reusable.

## Failure case

A redacted example of something that broke, drifted, harmed usability, or exposed a safety gap. Failure cases are contributions.

## Benchmark scenario

A reproducible task, input, metric, or expected behavior used to evaluate whether Jellyfish improved. Benchmark samples are contributions.

## Memory Crystal

A reusable structure distilled from repeated interactions or failures. It should include pattern, answer program, verifier, failure cases, provenance, and privacy status.

## RFC

A Request for Comments used for important protocol, governance, privacy, benchmark, or product direction changes.

## Public cache

A shared layer for reviewed and redacted structures. Private raw data must not enter the public cache.
