# CHARTER.md

## Mission

认知水母的使命是构建一个开源、可审计、可私有部署、可公共协作的残差操作系统，让人类愿望、失败、灵感、经验和 AI 能力共同形成智能循环。

## Nonprofit Protocol Layer

The protocol layer should remain neutral, auditable, and open:

- Residual Card Protocol
- Skill Capsule Protocol
- Proof of Compression
- Contribution Graph
- Shared Cache Object
- Human Transition Card
- Cognitive Fingerprint Rights

## Commercial Service Layer

Commercial services may provide:

- Hosted services
- Enterprise deployment
- Industry plugins
- Model routing
- Safety audits
- Team collaboration
- UI / App experiences

Commercial services must not privatize public protocols or the Public Skill Commons.

## Cognitive Commons

认知公地只接收脱敏、授权、验证、结构化的经验。原始隐私数据不得进入公共池。

## Human Transition Principle

不能用降低 `D_task` 的方式，制造更大的 `D_human`。

## Neutral Ecosystem Position

认知水母不是 OpenAI、Claude、DeepSeek、Gemini、Grok 或任何单一模型的附属品。它是模型之间、App 之间、用户之间、组织之间的中立层。

## Public Accountability

Shared skills, public cache objects, and canonical releases must preserve provenance, license status, review state, risk level, and rollback paths.

## Exit and Portability

Users and organizations should be able to keep memories private, export structured artifacts where appropriate, and leave hosted services without losing protocol-level rights.
